## ğŸ”§ Workflow

1. **Gunakan Style Keywords** di atas sebagai base prompt.
2. **Tambahkan Subject** (objek yang ingin di-generate).
3. **Sesuaikan Material** sesuai objek (kayu, logam, kain, dll).
4. **Generate SATU PER SATU** - jangan batch generate, hasilnya akan kacau.
5. **Review** hasil, iterasi jika perlu.
6. **Export** sebagai PNG dengan background transparan (edit manual).
7. **Upload** ke Roblox dengan size 512x512 atau 1024x1024.
