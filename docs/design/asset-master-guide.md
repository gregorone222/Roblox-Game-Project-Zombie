Dunia yang hancur namun indah. Alam mengambil alih, tapi survivor menciptakan kehangatan di antara reruntuhan.

### âœ… ALLOWED
| Category | Examples |
|:---------|:---------|
| Materials | Kayu, kain, logam berkarat, tanaman merambat |
| Tech | Radio tua, generator bensin, kabel warna-warni |
| Colors | Warm (sunset orange, forest green, wood brown) + accent (yellow, cyan) |
| Mood | "Cozy Apocalypse" - harapan di balik kehancuran |

### âŒ FORBIDDEN
| Category | What to Avoid |
|:---------|:--------------|
| Tech | Hologram, neon grid, laser, panel digital futuristik |
| Colors | Warna biru dingin steril, chrome/metal bersih |
| Entities | Alien, cyborg, teknologi sci-fi |

### Style Keywords
- **Stylized** - Not realistic
- **Cartoon** - Exaggerated proportions (Fortnite Style)
- **Vibrant** - Warm colors, fun & heroic

# ğŸ  ENVIRONMENT ART GUIDE

## Art Style Requirements
- **Reference**: Fortnite (Character & Furniture), Overwatch 2 (VFX Only)
- **Render**: Stylized polished 3D with soft gradients
- **Proportions**: Slight chunky/exaggerated

## Image Specifications
```
View: Isometric 3/4 angle
Background: Hitam
Lighting: Studio/dramatic warm lighting
```

## Asset Rules
1. **Furniture = Furniture SAJA** (tanpa batu, sarang laba-laba)
2. **Environment = Terpisah** (bisa di-compose di game)
3. **Style Konsisten** (Fortnite polished 3D - Chunky & Vibrant)

## Prompt Template
```
[Nama furniture] game asset, [kondisi rusak], post-apocalyptic 
abandoned aesthetic. Fortnite 3D style, game ready asset, warm 
sunset orange brown tones. Isometric 3/4 view, black background.
```

# ğŸ”« WEAPON ART GUIDE

## Skin Rarity Guidelines

| Rarity | Description |
|:-------|:------------|
| **Common** | Senjata dasar, warna standar, sedikit weathering |
| **Uncommon** | Skema warna custom, tape wrap, ukiran halus |
| **Rare** | Paint job unik, tekstur grip custom, minor 3D add-ons |
| **Epic** | Transformasi visual signifikan, tema spesifik |
| **Legendary** | Transformasi bentuk body senjata mengikuti tema |

## Per Weapon Type Rules

| Type | Rules |
|:-----|:------|
| Pistol | Compact, tanpa attachment, perban pada grip |
| SMG | Tanpa scope/rail eksternal, perban pada grip |
| Assault | Iron sights saja, full weapon visible |
| Shotgun | Heat shield boleh, tanpa attachment |
| Sniper | Scope sederhana, bipod dilipat/dihapus |
| LMG | Ammo belt di dalam drum, tanpa bipod |

## Forbidden Elements
- Efek hologram / neon grid
- Metal chrome bersih
- Render photorealistic
- VFX kompleks di gambar statis
- Sculpt tambahan besar

## Quality Checklist
- [ ] Palet warna proyek
- [ ] Style Fortnite (Chunky, Vibrant)
- [ ] Estetika weathered/makeshift
- [ ] Background bersih (hitam/transparan)
- [ ] Tidak ada VFX di gambar statis
- [ ] Full weapon visible
