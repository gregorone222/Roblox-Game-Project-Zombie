## Currency Types

| Currency | Type | Source | Usage |
|:---------|:-----|:-------|:------|
| **Coins** | Permanen | Wave Clear & Damage | Skin/Item permanen |
| **Points** | Sesi | In-game score | Buka pintu, beli senjata |
| **AP** | Permanen | Achievements | AP Shop items |
| **MP** | Permanen | Missions | MP Shop items |

## Coins Formula

**Base Ratio:** 20 Damage = 1 Coin

| Difficulty | Multiplier |
|:-----------|:-----------|
| Easy | 1.0x |
| Normal | 1.2x |
| Hard | 1.5x |
| Expert | 2.0x |
| Hell | 2.5x |
| Crazy | 3.0x |

## XP System

**Base Ratio:** 5 Damage = 1 XP

| Difficulty | Multiplier |
|:-----------|:-----------|
| Easy | 1.0x |
| Normal | 1.2x |
| Hard | 1.5x |
| Expert | 2.0x |
| Hell | 2.2x |
| Crazy | 2.5x |

## Progression Features

- **Leveling:** XP dari membunuh zombie
- **Leaderboards:** Global tracking (Kill, Damage, Level)
- **Daily Rewards:** Login bonus di Lobby
- **Titles:** Unlockable via achievements
